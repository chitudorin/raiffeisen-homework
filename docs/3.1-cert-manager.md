# cert-manager

For extra style points (and because I already like this setup I might as well use it for my homelab) I'll also set up cert-manager with a wildcard certificate.

I added an A record in the form of `*.home.domain.tld` pointing to my home network, will use cert-manager to issue a wildcard certificate and (hopefully) set up ingress-nginx to use this certificate automatically so I don't have to write it manually (the less YAML I have to write, the less chance I might mess something up).

I'm using the official documentation for installing with Helm and I can pretty much just copy the ingress-nginx setup to make sure everything's fine.

Turns out there is a difference compared to nginx-ingress and that's CRDs. The documentation mentions `--set crds.enabled=true` flag, this translates to:

```
spec:
  values:
    crds:
      enabled: true
```

After cert-manager is installed, I (manually, SOPS would also need a manual step for the private key so might as well create it this way) created a Secret containing my Cloudflare API key that is then used by the ClusterIssuer to issue a wildcard certificate:

```
---
apiVersion: cert-manager.io/v1
kind: ClusterIssuer
metadata:
  name: letsencrypt-prod
spec:
  acme:
    server: https://acme-v02.api.letsencrypt.org/directory
    email: dorin@chitzu.ro
    profile: tlsserver
    privateKeySecretRef:
      name: letsencrypt-prod
    solvers:
    - dns01:
        cloudflare:
          apiTokenSecretRef:
            name: cloudflare-api-token-secret
            key: api-token
---
apiVersion: cert-manager.io/v1
kind: Certificate
metadata:
  name: wildcard-home
  namespace: ingress-nginx
spec:
  secretName: wildcard-home-tls
  issuerRef:
    name: letsencrypt-prod
    kind: ClusterIssuer
  dnsNames:
  - "*.home.chitzu.ro"
  - "home.chitzu.ro"
```

Like I said earlier, I'd like nginx-ingress to use this certificate automatically for every Ingress, hence avoiding defining it every time, so I'll use this argument:

```
spec:
  values:
    controller:
      extraArgs:
        default-ssl-certificate: ingress-nginx/wildcard-home-tls
```
